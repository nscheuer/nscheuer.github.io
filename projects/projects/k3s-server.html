Built and maintain a home server running a lightweight Kubernetes distribution (K3S) on ARM-based hardware. Overcame compatibility challenges with certain workloads to deploy a variety of services, including:

  • MediaWiki – personal knowledge base for project and coursework documentation.
  • GitLab – used for CI/CD pipelines and automated deployments of personal projects.
  • GitLab Runner – deployed via Helm with automatic scaling based on demand.
  • Remote GPU pipeline – connection to my workstation, enabling offloaded AI tasks on a dedicated GPU.
  • Game servers – including Minecraft and BeamMP.
