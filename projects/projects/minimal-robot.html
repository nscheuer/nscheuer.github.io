I wrote this project as part of my bachelor's thesis at the Robotics Systems Lab at ETH Zurich. My project involved taking an existing quadruped robot called "Minimal" and adding wheels to enable it to perform hybrid walking/rolling locomotion. Specifically, my contributions were:

  • Evaluating different motor configurations and sizings for compatibility to the existing bus and mobility
  • Rapid prototyping for modified wheels and legs using 3D printing of PLA and TPU
  • Detailled modelling of the robot and actuators in URDF format
  • Training deep learning robust hybrid locomotion policies using Nvidia IsaacGym, validation of reference tracking in Gazebo
  • Rewriting the existing ROS control stack in ROS2 including modifications to account for new actuators and encoders
